# Titanic Survival Analysis – Power BI-style

## Overview
This project explores the Titanic dataset in two ways:  
1. Raw data exploration and pivot tables.  
2. Power BI-style dashboard using Google Sheets.

## Dataset
- File: `data/titanic.csv`  
- Contains passenger information: Age, Sex, Pclass, Fare, SibSp, Parch, Embarked, Survived.

## Analysis

### Perspective 1: Raw Data Exploration
- Created pivot tables in Google Sheets to understand survival patterns.  
- Focused on Gender, Passenger Class, Fare ranges, Embarked port, and Family Size.

## Key Insights
1. Females had higher survival rates than males.  
2. 1st class passengers survived more often than 2nd and 3rd class.  
3. Passengers paying higher fares had slightly higher survival rates.  
4. Small families (1–2 members) were more likely to survive than large families.
